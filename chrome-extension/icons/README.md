# Icon Files

The Chrome extension requires icon files in the following sizes:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## How to Create Icons

You can create these icons using any graphics editor (Figma, Photoshop, GIMP, etc.)

### Design Suggestion:
- Use the shield emoji (üõ°Ô∏è) or a simple shield shape
- Primary colors: Purple (#667eea) or Blue (#4299e1)
- Keep it simple and recognizable at small sizes

### Quick Generation:
You can use online tools like:
1. https://favicon.io/ - Generate icons from emoji or text
2. https://www.canva.com/ - Design custom icons
3. https://www.figma.com/ - Professional design tool

Place the generated PNG files in this `icons/` folder.

### Alternative (Temporary):
For testing purposes, you can use any 16x16, 48x48, and 128x128 PNG images.
The extension will work without icons, but Chrome will show a default plugin icon.
